## 🕹️ Como Jogar
1. Abra o arquivo `index.html` em qualquer navegador (ou utilize a extensão Live Server do VS Code).
2. No menu principal:
   * Pressione **[1]** para iniciar o jogo.
   * Pressione **[2]** para ver os créditos (Tela Sobre).
3. Controles do Jogo:
   * **Seta Esquerda / Seta Direita**: Movem a nave para os lados.
   * **[ESC]**: Volta para o menu principal a partir das telas de Sobre ou Game Over.
4. Objetivo: Desvie dos meteoros/inimigos vermelhos. A velocidade e a frequência deles aumentam a cada 100 pontos (troca de fase). Você tem 52 vidas.

## 🛠️ Tecnologias e Conceitos Utilizados
* **JavaScript & p5.js**
* **Orientação a Objetos**: Uso de classes para `Jogador` e `Inimigo`.
* **Vetores (`p5.Vector`)**: Movimentação e física dos elementos baseada em vetores.
* **Máquina de Estados**: Gerenciamento fluido de telas (Menu, Jogo, Sobre, Game Over).
* **Assets**: Gerenciamento de elementos visuais organizados.
